Detailed Description

ボタンに関する制御を提供します。

チャタリングの防止処理などが記載されており、ハードに近い部分のみ実装しています。
押された後のロジックに関しては別な部分で処理を行います。

ボタンの初期化処理を行います。

入力されたPin番号とボタン構造体を紐づけます。
例えば、構造体と5が入力された場合、構造体の状態はは5PINの入力で変化するようになります。

Parameters

pin: 入力PIN
debounceDelay: チャタリング防止のでバウンス時間
