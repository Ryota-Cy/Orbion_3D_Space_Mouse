Detailed Description

非同期的な遅延処理のためのヘッダファイルです。

このファイルでは、非同期ブロッキングを実現するために遅延状態を管理し、経過時間を確認するための関数を定義しています。

Function Documentation

◆ initDelay()

DelayState initDelay ( unsigned long  interval)

遅延の初期化を行います。

指定された遅延時間を設定し、初期状態として最後の実行時間を現在時刻で初期化します。

Parameters
interval: 遅延間隔（ミリ秒）を指定します。
Returns
DelayState : 初期化された遅延状態構造体を返します。

◆ isElapsed()

bool isElapsed ( DelayState &  state)

指定された遅延時間が経過したかを確認します。

現在の時間と最後に実行された時刻を比較し、指定された遅延間隔が経過していれば true を返し、次の実行時刻を更新します。

Parameters
state: 遅延状態を保持する構造体。
Returns
bool : 遅延時間が経過した場合は true、そうでない場合は false を返します。
